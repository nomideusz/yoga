# Project Charter

## Product

`szkolyjogi.pl` is a yoga school directory for Poland built on top of an existing calendar, booking, and QR entry system.

## Strategic Thesis

- Directory is the SEO + acquisition layer.
- Booking platform is the monetization core.
- Public discovery traffic should convert schools into SaaS customers.

## Primary Goals (Phase 1)

1. Launch a credible, searchable directory with ~500 prefilled schools.
2. Onboard initial showcase schools with real schedules.
3. Start repeatable outreach with a strong free-trial offer.
4. Validate school-to-SaaS conversion motion.

## Non-Goals (Phase 1)

- Expanding beyond yoga vertical.
- Building advanced CRM or multi-country support.
- Optimizing for enterprise custom contracts before product-market fit signals.

## Core Success Metrics

- Schools listed (total + claimed)
- Active schools using booking system
- Monthly bookings processed
- Trial-to-paid conversion rate
- Organic traffic to city/style/detail pages

## Core Positioning

For Polish yoga schools, `szkolyjogi.pl` is the easiest way to get discovered and manage bookings with modern UX, while preserving the option to keep their current website.
