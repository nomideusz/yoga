# Go-To-Market Plan

## Phase 1 — Launch the Directory

### Objectives

- Register and publish on `szkolyjogi.pl`.
- Seed initial supply using existing ~500 Google Places records.
- Establish trust with real showcase schools using live schedules.

### Key Actions

1. Register domain and set up production environment.
2. Import and normalize scraped schools into directory listings.
3. Onboard first case-study school (friend; permanently free).
4. Add 1–3 additional showcase schools with active bookings.

### Outputs

- Public directory with searchable city/style data.
- At least one live school detail page with real bookable classes.
- Initial case-study narrative and referral entry point.

## Phase 2 — Convert Schools to SaaS

### Offer

- 6-month free trial for booking system.
- Live demo as primary sales asset.
- Pricing aligned with Fitssey range to reduce switching resistance.

### Conversion Motion

1. Outreach to owners with concrete profile examples.
2. Demo: speed, UX, and real booking flow.
3. Month 4: show usage (bookings, revenue, engagement).
4. Introduce paid plans before trial end.

### Sales Messaging

- Modern, fast, and owner-friendly workflow.
- Zero/low migration friction via embeddable widget.
- Better student booking experience and reduced no-shows.

## ICP (Initial)

- Solo yoga teachers and small studio teams in Polish cities.
- Schools already receiving leads through Instagram/Google Maps.
- Operators struggling with manual booking over phone/messages.

## Channels

- Organic SEO from city/style pages.
- Founder-led outbound to local studio owners.
- Warm referrals via first teacher and community network.
