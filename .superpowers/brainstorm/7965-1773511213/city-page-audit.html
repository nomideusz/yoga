Issues Identified

1

Header hierarchy mismatch

Main page has: TAG → Title → Subtitle. City page has: Breadcrumb (WARSZAWA) → h1 (Warszawa) → subtitle. The breadcrumb acts as both navigation and tag, and the city name repeats redundantly.

2

No gradient / visual warmth

Main page has the radial ice gradient covering nav+hero. City page is flat white — feels disconnected from the brand.

3

Non-yoga styles showing (Stretching, Barre, etc.)

Main page filters these out. City page shows all styles including Stretching, Pilates Reformer, Barre, Tai Chi — should match the main page filtering.

4

Breadcrumb is minimal — no back link

Just shows "WARSZAWA" as current page. No "szkolyjogi.pl /" prefix for navigation back to main page.

5

Missing responsive breakpoints

Main page has min-height 900px, 1100px, 1400px breakpoints for tall screens. City page has empty @media block. No scaling for 2K/4K.

6

Style chips not horizontal-scroll on mobile

Main page pills scroll horizontally on mobile. City page style chips wrap, which can take a lot of vertical space.
